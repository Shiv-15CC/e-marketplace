# 🏛️ E-Marketplace Pro

A full-featured **E-Marketplace Web Application** built using **Streamlit**, **Oracle Database**, and **Python**.
Users can **buy, sell, wishlist items, receive alerts, and manage transactions** in a clean UI.

---

# 🚀 Features

## 👤 User Features

* User Registration & Login
* Browse Marketplace Listings
* Search & Filter by Category
* Buy Products
* Wishlist Items
* Notifications & Alerts
* Purchase History
* Reviews & Ratings

## 🛍️ Seller Features

* Post New Listings
* Manage Listings
* Delete Listings
* Track Sales

## 🔔 Smart Alerts

* Create Interest Rules
* Get Notifications
* Auto Match Listings

## 🛠️ Admin Features

* View Users
* Manage Categories
* View Transactions
* Platform Analytics

---

# 🧰 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python

  * pandas

  * oracledb

# ⚙️ Installation

## 1. Clone Repository

```
git clone https://github.com/Shiv-15CC/e-marketplace.git
```

## 2. Install Dependencies

```
pip install streamlit pandas oracledb
```

## 3. Run Application

```
streamlit run app.py
```

---

# 🗄️ Database Setup

Make sure Oracle XE is running:

```
dsn="localhost/XE"
```

Update credentials in `app.py`:

```
user="system"
password="your_password"
```
